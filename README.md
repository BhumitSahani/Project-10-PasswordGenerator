# Project-10-PasswordGenerator
This is the nineth project of my Web Development journey. This project allows the user to generate random passwords using all the characters in ASCII code. One of it's functionality, that I added without instructor help, allows user to choose whether they want the password to be alphanumeric, alphabetic or digits only along with selecting length.
