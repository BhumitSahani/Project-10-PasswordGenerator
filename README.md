# Project-10-PasswordGenerator
This is the nineth project of my Web Development journey. This project allows the user to generate random passwords using all the characters in ASCII code. One of it's functionality allows user to choose whether they want the password to be alphanumeric, alphabetic or all ASCII characters only along with selecting length.
